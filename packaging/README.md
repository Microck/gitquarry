# Packaging

Release-aligned packaging files for `gitquarry` live here.

- `homebrew/gitquarry.rb` mirrors the public Homebrew tap in `Microck/homebrew-gitquarry`
- `scoop/gitquarry.json` mirrors the public Scoop bucket in `Microck/scoop-gitquarry`
- `aur/` contains Arch packaging files for local `makepkg` use or future AUR publication

Update these files whenever a GitHub release version or hash changes.
